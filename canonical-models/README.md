# Canonical Models for Core Lending Modernization

This folder contains canonical data models used across the lending lifecycle.  
These models provide:

- A unified semantic layer across legacy and modern systems  
- Consistent event and API payload structures  
- A foundation for domain-driven design  
- A shared vocabulary for origination, underwriting, servicing, and collections  

Each model is intentionally minimal, extensible, and aligned with event-driven modernization patterns.
