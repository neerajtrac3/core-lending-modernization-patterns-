# Legacy Complexity Index (LCI)

The Legacy Complexity Index (LCI) is a quantitative scoring model that measures how difficult it is to modernize a legacy system.  
It evaluates six dimensions (1–5 each):

1. Integration Density  
2. Codebase Complexity  
3. Customization Level  
4. Documentation Quality  
5. Data Fragmentation  
6. Operational Dependencies  

**LCI Score Bands**
- 1–10: Low → Rehost / Replatform  
- 11–18: Medium → Refactor  
- 19–24: High → Rearchitect  
- 25–30: Critical → Replace / Retire  

LCI provides an objective, defensible basis for modernization sequencing.
