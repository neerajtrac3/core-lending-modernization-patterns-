# Event Taxonomy

Events across the lending lifecycle are classified into five categories:

1. State Events  
   - LoanCreated, PaymentPosted  

2. Decision Events  
   - UnderwritingDecisionMade  

3. Lifecycle Events  
   - LoanFunded, LoanClosed  

4. Exception Events  
   - FraudFlagged, BrokenPromise  

5. Reference Events  
   - CustomerUpdated, ProductChanged  

This taxonomy ensures consistent event modeling across domains.
