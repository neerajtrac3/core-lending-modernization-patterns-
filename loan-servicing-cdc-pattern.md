# Loan Servicing CDC Pattern

## Problem
Legacy servicing systems cannot emit real‑time events.

## Solution
Use Change Data Capture (CDC) to publish servicing updates as canonical events.

## CDC Triggers
- Payment posted  
- Delinquency status changed  
- Escrow adjustment  
- Payoff initiated  

## Benefits
- Real‑time servicing updates  
- Enables C360, collections, and risk analytics  
- No changes required to legacy core
